# select 选择器


## 协议

```json
{
  "type": "",
  "title": "",
  "value": "",
  "eventlist": []
}
```

## 协议属性
| 属性名称 | 说明 | 取值类型 | 默认值
| ---- | ---- | ---- | ---- |
| eventlist.trigger | 事件钩子 | enum |  |



### eventlist.trigger
| 值 | 说明 |
| ---- | ---- |
| onload | 加载时触发 |
| onvaluechange | 值改变时触发 |
