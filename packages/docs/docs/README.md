## @smart100/web-engine 开发文档

> 玄迅销售云智慧100 H5 端低代码应用。

## 概述

@smart100/web-engine 是一个基于 Vue2 + Element-UI 的通用页面协议引擎，它可以帮助你使用协议快速生成页面。

查看[快速开始](introduction/quickstart)了解详情。

## 特性

- 基于 Vue2 + element-ui
- 入门简单 快速生成页面
