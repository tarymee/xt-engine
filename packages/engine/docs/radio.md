# radio
单选控件

## 示例
暂无

## 属性说明

```json
{
  "type": "radio",
  "code": "radio-1",
  "title": "单选控件",
  "value": "",
  "displaytype": "",
  "options": [
    {
      "key": "1",
      "text": "启用"
    },
    {
      "key": "0",
      "text": "停用"
    }
  ],
  "eventlist": [
    {
      "trigger": "onvaluechange",
      "handler": ""
    },
    {
      "trigger": "onload",
      "handler": ""
    }
  ]
}
```

### displaytype 样式

选项值排列方式，默认值为 horizontal

| 值          | 说明                       |
| ----------- | -------------------------- |
| horizontal  | 常规，默认值，选项值水平排列 |
| vertical    | 选项值垂直排列              |

### value 默认值
字符串单值类型
